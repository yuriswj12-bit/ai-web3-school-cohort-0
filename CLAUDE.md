# CLAUDE.md

## 项目概述

AI x Web3 School Cohort 0 个人学习仓库。用于学习日志、任务证明、代码实验、Handbook 反馈和 Hackathon 准备。

## 目录结构

- `daily/YYYY-MM-DD.md` — 每日学习笔记
- `tasks/` — 任务记录
- `experiments/` — 代码实验与原型
- `curriculum/weekN.md` — 课程大纲本地化
- `handbook-feedback/` — Handbook 反馈
- `hackathon/` — Hackathon 项目准备
- `submissions/` — 任务提交记录
- `scripts/` — 自动化脚本

## 工作规范

- 默认用中文
- 这是 public 仓库，绝对不要写入 API Key、私钥、助记词、token
- commit/push 前人工确认
- 每日笔记按模板填写
