# AI × Web3 School 学习仓库

本仓库用于记录我在 AI × Web3 School 的学习过程、任务证明、实验记录、每日打卡和 Handbook 反馈。

## 仓库用途
- 个人学习日志
- 任务证明与实验记录
- 每日打卡草稿
- Handbook feedback
- Hackathon 项目准备

## 资源链接
- Handbook：https://aiweb3.school/zh/handbook/
- WCB 课程：https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning：https://web3career.build/zh/programs/AI-Web3-School#tab=learning

## Week 1 学习目标与记录结构
本周主要目标是完成仓库初始化、建立标准学习记录流程，并开始系统阅读 Handbook。

记录结构说明：
- `daily/`：每日学习打卡与复盘
- `tasks/`：待办任务清单
- `experiments/`：代码实验与实践记录
- `handbook-feedback/`：Handbook 问题、建议与反馈
- `learning-plan.md`：个人学习计划
- `profile.md`：个人背景

## 使用 Learning Agent 初始化说明
本仓库由 AI × Web3 School Learning Agent 辅助初始化。

Agent 完成的工作：
- 引导 GitHub CLI 安装与登录
- 创建标准目录结构
- 生成 profile.md、daily-note 模板和第一天打卡记录
- 协助清理旧文件并提交

我人工确认的内容：
- 学员画像（一位流动性交易者，每日投入 3 小时以上）
- 仓库名称与可见性
- 所有删除操作、文件写入和 commit 内容
- 最终提交前的内容检查
