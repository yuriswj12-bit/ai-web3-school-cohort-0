# ai-web3-school-cohort-0

AI x Web3 School Cohort 0 个人学习仓库。

## 这是什么仓库？

用于记录 AI x Web3 School 的学习过程、任务证明、代码实验和 Hackathon 准备。

## 我该做什么？

1. 每天写学习笔记 → `每日笔记/YYYY-MM-DD.md`
2. 记录任务完成情况 → `任务/`
3. 做代码实验 → `实验/`
4. 同步课程大纲 → `课程大纲/`
5. 提交 Handbook 反馈 → `手册反馈/`
6. 准备 Hackathon → `黑客松/`

## 工作规范

全部规则和流程见 [CLAUDE.md](CLAUDE.md)

## 目录结构

- 每日笔记/     — 每天的学习记录
- 任务/         — 任务完成证明
- 实验/         — 代码实验和原型
- 课程大纲/     — WCB 课程同步
- 手册反馈/     — Handbook 反馈
- 黑客松/       — Hackathon 项目
- 提交记录/     — 任务提交历史
- 脚本/         — 自动化工具
