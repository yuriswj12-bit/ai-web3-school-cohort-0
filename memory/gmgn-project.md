### 3. 量化交易策略（RSI择时）
来源：擒龙术一

**买入条件**：
- RSI跌破30 → 观望等待成交量减少
- RSI<20 + 成交量增加 + 阳线 → 抄底买入（分批）

**卖出条件**：
- RSI突破70 + 成交量增加 → 分批止盈
- RSI突破70 + 成交量减少 → 清仓

### 4. 找庄方法
来源：擒龙术二、擒龙术三

**核心方法**：
- 庄家地址成"组"：多个钱包同时同时间段交易，金额相近
- 洗盘识别：砸出的筹码短时间内被其他地址买回
- 庄家跑了：持仓排名上升 + 大额筹码流出
- 短期热点：有庄的表现是前排地址有超高倍数+盈利且没走

### 5. 庄家坐庄全流程
来源：擒龙术四

**庄家特征识别**：
- 拉盘特征：大额买单（>1000刀）连续买入
- 吸筹特征：最早的交易记录，庄家狙击底筹
- 庄家拿筹码：通常在部署后几分钟内通过多个地址拿走70%+筹码

**典型流程（LILX案例）**：
1. 部署代币 → DEV号拿34%筹码
2. 3分钟内 → 13个地址狙击，拿走42%筹码
3. 内盘76%筹码被庄家控制
4. 添加流动性 → 庄家分散派发给71个地址+3位KOL
5. 拉盘吸引注意力 → 极速拉升
6. 获利了结 → 砸盘

**流动性与走势判断**：
- 流动性好 → 庄家可能格局更大，拉更高
- 流动性差 → 庄家可能快速砸盘走人

### 6. 打新盘秘法
来源：擒龙术五

**核心逻辑**：
- MEME是击鼓传花游戏
- 要抢在绝大多数人之前发现并买入
- 不能跟公开的聪明钱地址（那些是大部分人跟的）
- 赚钱逻辑：把货倒给后来的人，而不是抬轿子

**买之前问自己**：谁来接我的盘？

**正确姿势**：
- 成为"扫链隐身高手" - 最早发现新盘
- 抢在KOL和公开地址之前买入

### 7. 小资金起步策略
来源：擒龙术六

**四种策略模型（数学分析）**：
1. 出手次数多+追求高利润 → 亏损概率99%，不适合小资金
2. 出手次数多+追求低利润 → 较难，依赖喊人接盘
3. 出手次数少+追求高利润 → 对经验要求极高
4. **出手次数少+追求低利润** → **最优解！**

**最优策略（少出手+低利润）**：
- 出手次数：1-3次
- 止盈点：25%-100%
- 每次金额：0.5 SOL
- 100K-5M市值区间盈利概率：99.67%

**关键数据**：
- Pump 4/15生成43,271个代币，毕业（打满发射）仅408枚
- 毕业率：0.94%
- 内盘买入后翻倍需要喊约4000U买盘

**结论**：小资金起步最佳策略是少出手+低利润率

### 8. 链上BUG与赚钱本质
来源：擒龙术七

**核心观点**：
- 市场交易 = 打游戏
- 普通人赚钱的唯一方法：找到BUG并利用它

**BUG的定义**：
1. 能迅速且稳定赚钱
2. 知道的人少
3. 路径简单，可重复
4. 发现后觉得是脑筋急转弯

**实践方法**：
- 找到BUG → 疯狂牟利
- BUG被更多人知道后市场会修复
- 修复前如果利益不够退休 → 继续找下一个BUG
- 在找到下一个BUG前，不要动一分钱

**重要认知**：
- 99.99%的人不适合交易
- 如果你适合交易，从第一天开始就能赚到钱
- 普通人要抛弃"成为技术交易员"的幻想
