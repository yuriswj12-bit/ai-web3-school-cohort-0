# Week 1 Web3 概念翻转卡片 Demo 记录

今天把 Web3 基础概念整理成了可交互网页。

文件在 scripts/web3-concept-cards.html

双击就能打开。12 张卡片，正面只显示概念名。点击翻转，背面显示解释、例子、安全提醒。

做这个 Demo 的原因很简单。Task 3 要求整理概念，我不想只交一篇 markdown。我想做点能实际点开的东西。翻转卡片形式最直接，点一下就能看到内容，符合我对学习工具的要求。

Demo 完全本地运行。没有后端。没有数据库。Tailwind 用 CDN，打开即用。任何人都能直接用浏览器打开仓库里的这个文件。

我把 13 个概念全放进去了。每个概念都写了三部分：解释、例子、安全提醒。重点把私钥、助记词、签名、授权这几个高风险操作单独做了提醒。

这个 Demo 同时能当 Task 1 的可交互学习产物提交。交互方式简单，点击翻转。AI 只负责生成概念内容，翻转效果和样式都是我自己调整的。

下一步如果要做进阶版，可以把这些概念做成 Anki 卡片，或者接真实链上数据做动态卡片。但当前这个版本已经满足作业要求。
